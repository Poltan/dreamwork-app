# Dreamwork — resume-to-jobs prototype

Загрузка резюме + пожеланий → подбор вакансий напрямую с сайтов работодателей по всему миру
(включая Россию, США, Канаду, Израиль, Саудовскую Аравию, ОАЭ и др.) → кнопки «написать сопроводительное письмо»
и «скопировать + отклик» (переход на страницу отклика работодателя/ATS).

Это рабочий прототип (v1, end-to-end). Без регистрации и оплаты — один экран.

## Как это работает

```
резюме (PDF/DOCX/TXT) + страна + зарплата
        │
        ▼
 [бэкенд] разбор резюме ──► Claude извлекает профиль (роль, грейд, навыки, цели, языки)
        │
        ▼
 поиск по job-API (гибрид): hh.ru (RU) · Adzuna (US/CA/UK/EU/AU…) · Jooble (глобально, вкл. IL/SA/AE) · Remotive (remote)
        │
        ▼
 Claude ранжирует и пишет fit-заметку под кандидата ──► карточки + письма
```

«Напрямую от работодателя»: ссылка отклика в каждой карточке ведёт на страницу вакансии работодателя/ATS,
которую отдаёт провайдер. Кнопка «скопировать + отклик» кладёт письмо в буфер и открывает эту страницу.

## Структура

```
dreamwork-app/
  backend/
    app.py            FastAPI: /api/match, /api/letter, /api/health, "/" (отдаёт фронтенд)
    providers.py      провайдеры вакансий + маршрутизация по стране + дедуп
    requirements.txt
    .env.example      сюда вписать ключи (скопировать в .env)
  frontend/
    index.html        одностраничный интерфейс (стиль DreamWork)
```

## Запуск локально

```bash
cd dreamwork-app/backend
python -m venv .venv && source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env          # и вписать ключи (см. ниже)
uvicorn app:app --reload --port 8000
```

Открыть в браузере: **http://localhost:8000** (фронтенд отдаётся самим бэкендом).

## Какие ключи нужны (всё на бесплатных тарифах)

| Что | Зачем | Где взять | Обязателен |
|---|---|---|---|
| `ANTHROPIC_API_KEY` | разбор резюме, ранжирование, письма | console.anthropic.com | **да** |
| `JOOBLE_API_KEY` | глобальный поиск, вкл. Израиль/Саудовскую Аравию/ОАЭ | jooble.org/api/about | желательно |
| `ADZUNA_APP_ID` + `ADZUNA_APP_KEY` | зарплаты по US/CA/UK/EU/AU и др. | developer.adzuna.com | опционально |
| hh.ru, Remotive | Россия/СНГ и remote | — | без ключа |

Минимум для старта: только `ANTHROPIC_API_KEY` — тогда работают hh.ru (Россия) и Remotive (remote).
Для США/Израиля/Саудовской Аравии и т.д. добавьте Jooble (и/или Adzuna).

## Деплой (когда прототип устроит)

Любой хостинг для Python-веба: Render / Railway / Fly.io. Команда запуска:
`uvicorn app:app --host 0.0.0.0 --port $PORT`, переменные окружения — те же, что в `.env`.
Фронтенд отдаётся тем же сервисом, отдельный хостинг не нужен.
